# TikTok KOL Sourcing V5

> 一款让 TikTok 红人建联更高效的工具 🎯

---

## 它能做什么

### 找达人

用关键词搜 TikTok 账号，出来的全是商家号。这个工具换了个思路——**搜视频，再提取创作者**。

输入一个亚马逊链接，AI 会自动分析产品，生成四组关键词：

| 类型 | 作用 |
|------|------|
| 品类词 | 找已经在做这类内容的博主 |
| 场景词 | 找目标用户常看的视频类型 |
| 竞品词 | 找已经推过竞品的博主（转化率最高） |
| 人群词 | 扩大范围，覆盖更多潜在人选 |

### 筛达人

采回来的数据，系统按信号打分：

- 有邮箱的优先（能直接联系）
- 粉丝量符合预期的优先（量级匹配）
- Bio 里提到相关品类的优先（内容相关）
- 推过竞品的优先（已经被市场验证）
- Bio 里写明接受合作的优先（回复率高）

总分从高到低分成 A/B/C 三级。A 级是优先联系对象。

### 发邮件

邮件不是替换个名字就完事，而是**引用博主的具体视频内容**：

> "看到你那款 Litter Robot 4 的测评视频，2.7M 播放，评论区都在问好用吗——你的粉丝对这个品类有真实需求。"

根据博主规模自动调整语气：小博主轻松随意，大博主专业简洁。

### 看结果

输出三样东西：
- CSV 表格（数据存档）
- HTML 报告（直接在浏览器打开，支持筛选排序）
- 开发信文本文件（可直接复制使用）

## 快速开始

```bash
git clone https://github.com/All-cookies/TikTok-KOL-Sourcing-V5.git
cd TikTok-KOL-Sourcing-V5
cp .env.example .env
# 填入 TikHub API Key
npm install
```

在 Claude Code 中输入 `搜集小助手启动`，剩下的对话完成。
