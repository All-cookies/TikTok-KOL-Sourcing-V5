# TikTok KOL Sourcing V5

> 让 TikTok 红人建联从"大海捞针"变成"精准打击"

---

## 这个工具解决什么问题

找 TikTok 达人，你还在这样操作吗？

- 手动搜关键词 → 出来的全是商家号和搬运号
- 一个一个看 Bio → 花半小时才筛出 5 个能联系的
- 发邮件过去 → 石沉大海，没人回复

V5 版本换了思路：**先搜视频，再从视频里提取创作者**。配合 AI 自动分析，帮你生成最精准的关键词组合。

---

## 核心功能

### 1. AI 关键词生成

丢一个亚马逊产品链接，AI 自动分析并生成四组关键词：

| 类型 | 作用 | 示例 |
|------|------|------|
| 品类词 | 找已经在做这类内容的博主 | "cat litter review" |
| 场景词 | 找目标用户常看的视频类型 | "morning routine with pets" |
| 竞品词 | 找推过竞品的博主，转化率最高 | "Litter Robot review" |
| 人群词 | 扩大范围，覆盖更多潜在人选 | "pet mom essentials" |

**为什么竞品词最重要？** 推过竞品的博主，意味着他的粉丝对这个品类有真实需求，转化路径已经被验证过了。

---

### 2. 智能评分系统

采集回来的达人按信号打分，优先级一目了然：

| 加分项 | 分值 | 为什么重要 |
|--------|------|------------|
| 有邮箱 | +30 | 能直接联系，不用绕路 |
| 粉丝量符合预期 | +20 | 量级匹配，不过高也不过低 |
| Bio 提到相关品类 | +15 | 内容本来就跟品牌相关 |
| 来自竞品词搜索 | +15 | 竞品已验证过这批用户的转化能力 |
| Bio 有 "collab/PR" | +10 | 明确写着接受合作，回复率高 |
| 活跃创作者 | +10 | 持续产出内容，不是僵尸号 |

**A级达人（60+分）**：优先联系，直接发邮件
**B级达人（40-59分）**：先关注互动，养熟再建联
**C级达人（<40分）**：备选观察，等时机

---

### 3. 个性化邮件生成

邮件不是替换个名字就完事，而是**引用博主的具体视频内容**，让对方觉得"这封邮件是专门写给我的"。

> "看到你那款 Litter Robot 4 的测评视频，2.7M 播放，评论区 3000 条都在问好用吗——你的粉丝对这个品类有真实需求。"

还会根据博主规模自动调整语气：

| 博主类型 | 语气风格 |
|----------|----------|
| 小博主（<10K） | 轻松随性，像朋友一样聊天 |
| 中型博主（10K-100K） | 友好专业，展示诚意 |
| 大博主（100K+） | 专业简洁，节省双方时间 |

---

### 4. 三种输出格式

| 格式 | 用途 |
|------|------|
| CSV 表格 | 数据存档，导入 CRM |
| HTML 报告 | 浏览器打开，运营快速筛选判断 |
| 开发信文本 | 批量生成的个性化邮件，直接复制使用 |

---

### 5. 可视化报告

打开 HTML 报告，无需打开 Excel，所有信息一目了然：

- 基本信息 — 账号、粉丝数、评级
- 联系方式 — 邮箱、主页链接
- Bio 内容 — 直接判断风格是否匹配
- 匹配视频 — 点开直接看，可跳转到 TikTok
- 关键词来源 — 知道是通过哪个词找到的

## V5 vs V6Web

| 功能 | V5 | V6Web |
|------|-----|-------|
| AI 关键词生成（亚马逊链接） | 支持 | 不支持 |
| 竞品词/场景词/人群词分类 | 支持 | 不支持 |
| 个性化邮件自动生成 | 支持 | 不支持 |
| HTML 可视化报告 | 支持 | 不支持 |
| 多种输出格式 | CSV + HTML + 邮件 | CSV |
| 评分维度 | 6 维度 | 9 维度（更细） |
| 暗色模式 | 不支持 | 支持 |
| 多关键词标签输入 | 不支持 | 支持 |
| 达人评级筛选 | A/B/C | A/B/C |

**选 V5 如果你想要**：AI 辅助分析 + 邮件生成 + 一步到位的完整工作流

**选 V6Web 如果你想要**：更细致的评分体系 + 更现代的 UI + 纯筛选建联

## 快速开始

```bash
git clone https://github.com/All-cookies/TikTok-KOL-Sourcing-V5.git
cd TikTok-KOL-Sourcing-V5
cp .env.example .env
# 编辑 .env，填入 TikHub API Key
npm install
```

启动后，在 Claude Code 中输入 `搜集小助手启动`，剩下的交给对话。
