# TikTok KOL Sourcing V5

> 一款让 TikTok 红人建联更高效的工具。

---

## 场景

周一早上，需要找一批宠物用品的 TikTok 达人。

把亚马逊链接丢给 AI，自动分析产品和竞品，生成关键词策略，一键采集红人数据。采集完自动评分分级，还能批量生成个性化开发信。

整个过程 20 分钟左右，不用切换多个工具，不用手动整理 Excel。

---

## 核心能力

### 视频搜索

搜关键词找账号，商家号太多。这个工具搜的是视频——先找到内容，再提取创作者。配合四维关键词策略（品类词 + 场景词 + 竞品词 + 人群词），找到的都是真正在拍内容的博主。

### 自动评分

采集完的达人会自动打分：

| 加分项 | 分值 |
|--------|------|
| 有邮箱 | +30 |
| 粉丝量在目标区间 | +20 |
| Bio 里提到相关品类 | +15 |
| 来自竞品词搜索 | +15 |
| Bio 里有 "collab / PR" 字样 | +10 |
| 活跃创作者 | +10 |

- A 级（60+）直接发邮件
- B 级（40-59）先关注互动
- C 级（<40）备选

### 个性化邮件

邮件会引用博主的具体内容，不是简单的名字替换：

> "看到你那款 Litter Robot 4 的测评视频，2.7M 播放，评论区都在问好用吗——你的粉丝对这个品类有真实需求。"

根据博主规模自动调整语气。小博主轻松随性，大博主专业简洁。

### 可视化报告

HTML 报告包含：
- 关键词来源分布
- A/B/C 分级一览
- 每位达人的 bio、邮箱、代表作视频
- 支持筛选、搜索、排序

---

## 工作流

```
产品链接 → 关键词策略 → 数据采集 → 评分分级 → 开发信生成
```

1. 丢一个亚马逊链接，自动提取产品信息
2. AI 推荐关键词组合
3. 自动去重 + 评分 + 分级
4. 一键生成个性化邮件

## 快速开始

```bash
git clone https://github.com/All-cookies/TikTok-KOL-Sourcing-V5.git
cd TikTok-KOL-Sourcing-V5
cp .env.example .env
# 填入 TikHub API Key
npm install
```

在 Claude Code 中输入 `搜集小助手启动`，剩下的对话完成。
