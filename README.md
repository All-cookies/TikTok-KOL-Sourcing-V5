# TikTok KOL Sourcing V5

> 不只是采集工具，而是你完整的 TikTok 红人建联工作台。

---

## 我是谁 & 为什么做了这个项目

我一直在做跨境出海，最痛的一件事就是 **TikTok 红人建联**。

市面上的红人工具要么太贵，要么采集了数据还得自己一条条写邮件。这个项目是我自己用来解决这个问题的——从产品分析 → 达人发现 → 筛选评分 → 开发信生成，一条龙自动化完成。

它不是一个大平台，而是一个 **AI Agent Skill**，你装了之后直接在终端里和 AI 对话就能完成整条工作流。

---

## 红人营销的痛点 & 我们怎么解决

### 痛点 1：找到的人不匹配

> 搜关键词出来一堆商家号、搬运号，真正的内容创作者淹没在数据里。

**我们的方案**：不做用户搜索，而是做 **视频搜索**。搜视频，再从视频提取创作者。这样找到的一定是真的在做内容的博主，而不是挂着关键词的营销号。

另外，我们用 **四维关键词模型**：
- 品类词 → 搜已经在做这个品类内容的
- 场景词 → 搜目标人群常看的生活场景
- 竞品词 → 搜已经在推广竞品的（转化率最高）
- 人群词 → 扩大漏斗触达泛人群

### 痛点 2：采集了一堆数据，不知道怎么排优先级

> 上百个达人，谁先联系？谁值得深入合作？

**我们的方案**：一套 **评分体系**，不是凭感觉，而是按信号打分：

| 信号 | 分值 | 业务逻辑 |
|------|------|----------|
| 有邮箱 | +30 | 能直接联系，不需要再绕路 |
| 粉丝在目标区间 | +20 | 粉丝量级匹配，不会大材小用 |
| Bio 含品类词 | +15 | 内容本身就相关，不是硬推 |
| 来自竞品搜索 | +15 | 竞品已经验证过的流量池 |
| Bio 含"pr / collab" | +10 | 明确开放合作，回复率更高 |
| 活跃创作者 | +10 | 持续产出，不是弃号 |

最后输出 **A/B/C 三级**，A 级直接发开发信，B 级先互动，C 级备选。

### 痛点 3：开发信写了等于没写

> "Dear influencer, we love your content, please promote our product."——这种模板邮件谁会回？

**我们的方案**：每封开发信都是 **个性化的**。

不是替换个名字就叫个性化，而是：
1. **引用博主的具体内容**（"看到你那条 Litter Robot 的测评，2.7M 播放，很厉害"）
2. **匹配语气**：小博主亲切随性，大博主专业简洁
3. **五段式结构**：钩子 → 自我介绍 → 价值主张 → CTA → 署名

```
Subject: 沙琪玛，你的 Litter Robot 测评太棒了！

Hi 沙琪玛，

看到你那款 Litter Robot 4 的开箱视频，2.7M 播放量，
评论区 3000 多条都在问"好用吗"，说明你的粉丝对这个品类有真实需求。

我们做的是 PETKIT PuraMax 2 自动猫砂盆（$389），
核心卖点是 360° 防漏 + App 智能控制，
和你现有的 Litter Robot 内容天然互补。

如果是你感兴趣的话，我们可以提供样品 + 专属折扣码，
你的粉丝还能享受额外优惠。

你觉得怎么样？

Cheers
[你的名字]
```

### 痛点 4：采了一堆数据还要去 Excel 里筛

> CSV 导出了，然后呢？一个个看？

**我们的方案**：直接生成 **可视化 HTML 报告**，不用打开 Excel。

```
┌─────────────────────────────────────────────────┐
│ 总博主数 50    A 级优先 30    邮箱获取率 80%  │
├─────────────────────────────────────────────────┤
│ 关键词标签画像：cat mom routine ████████████ 21  │
│                Litter Robot review ██████████ 19 │
├─────────────────────────────────────────────────┤
│ #1 [A] @_brittanylc          20.7K粉  2.7M播放  │
│     📧 brittanylc@...  |  🔗 bio.site/...       │
│     Bio：🐱 Cat mom | 📍 NYC | 📧 collab@...    │
│     matched video：This is your sign to... ▶ 观看 │
└─────────────────────────────────────────────────┘
```

筛选搜索、按评级过滤、按播放量排序，全部浏览器里完成。

## 快速上手

### 1. 安装

```bash
cp .env.example .env
# 填入你的 TikHub API Key
npm install
```

### 2. 启动

在 Claude Code 中输入：

```
搜集小助手启动
```

然后跟 AI 对话就行，像这样：

```
你：帮我找一批美区宠物智能用品的达人
AI：请提供商品链接或产品描述

你：https://www.amazon.com/...（PETKIT 自动猫砂盆 $389）
AI：✅ 产品分析完成
    请确认筛选偏好：粉丝区间？目标市场？互动率？

你：3K-200K 粉丝，美区，互动率 3% 以上
AI：开始采集... ✅ 找到 47 位达人
    需要生成开发信吗？
```

### 3. 输出

- `output/*.csv` — 结构化红人数据
- `output/*.html` — 可视化报告
- `output/emails-*.txt` — 个性化开发信
