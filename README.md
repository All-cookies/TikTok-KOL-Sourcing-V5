# TikTok KOL Sourcing V5

> 一款让 TikTok 红人建联更高效的工具 🎯

---

## 场景

周一早上，需要找一批宠物用品的 TikTok 达人合作。

把亚马逊链接丢给 AI，自动分析产品和竞品，生成关键词策略，一键采集红人数据。采集完自动评分分级，还能批量生成个性化开发信。

整个过程 20 分钟左右，不用切换多个工具，不用手动整理 Excel。

---

## 核心能力

### 视频搜索

关键词搜账号，出来的全是商家号和搬运号。

这个工具换了个思路——**搜视频，再提取创作者**。先找到正在做某品类内容的视频，再从视频里找到发布者。这样找到的每一个都是真正在拍东西的真人。

配合四维关键词策略：

- **品类词** → 搜已经在做这个品类内容的博主
- **场景词** → 搜目标人群常看的生活场景
- **竞品词** → 搜已经在推竞品的博主（转化率最高）
- **人群词** → 扩大漏斗，触达泛人群

### 自动评分

采集回来的达人，系统会打分排序：

| 加分项 | 分值 | 为什么 |
|--------|------|--------|
| 有邮箱 | +30 | 能直接联系，不用绕路 |
| 粉丝量在目标区间 | +20 | 量级匹配 |
| Bio 里提到相关品类 | +15 | 内容本来就跟品牌相关 |
| 来自竞品词搜索 | +15 | 竞品已经验证过 |
| Bio 里有 "collab / PR" 字样 | +10 | 明确开放合作 |
| 活跃创作者 | +10 | 不是弃号 |

根据总分分成三级：
- **A 级（60+）** — 直接发邮件
- **B 级（40-59）** — 先关注互动
- **C 级（<40）** — 备选观察

### 个性化邮件

模板邮件大家都在用，回复率越来越低。

这套工具生成的邮件会**引用博主的具体内容**：

> "看到你那款 Litter Robot 4 的测评视频，2.7M 播放，评论区 3000 条都在问好用吗——你的粉丝对这个品类有真实需求。"

不止替换名字，还会：
- 引用博主近期视频的具体数据和内容
- 根据博主规模调整语气（小声博亲切随性，大博主专业简洁）
- 提供具体的合作价值（样品、折扣码、专属福利）

### 可视化报告

打开 HTML 报告，可以看到：
- 关键词来源分布（哪个词找到的人最多）
- A/B/C 分级一览
- 每位达人的 bio、邮箱、代表作视频链接
- 支持筛选、搜索、按评级/粉丝/播放量排序

## 工作流

```
产品链接 → 关键词策略 → 数据采集 → 评分分级 → 开发信生成
```

1. 丢一个亚马逊链接，AI 自动提取产品信息、竞品、卖点
2. AI 推荐关键词组合，确认后开始采集
3. 自动去重 + 评分 + 分级
4. 一键生成个性化开发信

## 快速开始

```bash
git clone https://github.com/All-cookies/TikTok-KOL-Sourcing-V5.git
cd TikTok-KOL-Sourcing-V5
cp .env.example .env
# 填入 TikHub API Key
npm install
```

在 Claude Code 中输入 `搜集小助手启动`，剩下的对话完成。
