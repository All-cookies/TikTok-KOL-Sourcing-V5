## 🚀 核心能力

### 1️⃣ 视频搜索，而非账号搜索

关键词搜账号，一搜一个商家号 😑

这个工具搜的是**视频**——先找到内容，再提取创作者。找到的每一个都是真正在拍东西的真人 📹

再配合四维关键词策略（品类词 + 场景词 + 竞品词 + 人群词），漏斗精准得多 🎯

### 2️⃣ 自动评分，不用瞎猜优先级

采集回来的达人，系统会打分：

| 加分项 | 分值 |
|--------|------|
| 有邮箱 📧 | +30 |
| 粉丝量在目标区间 | +20 |
| Bio 里提到相关品类 | +15 |
| 来自竞品词搜索 | +15 |
| Bio 里有 "collab / PR" 字样 | +10 |
| 活跃创作者 | +10 |

- 🏆 **A 级（60+）** — 直接发邮件
- 🔔 **B 级（40-59）** — 先关注互动
- 📦 **C 级（<40）** — 备选

### 3️⃣ 个性化邮件，不是"Dear {{name}}"

模板邮件的回复率大家都懂 😅

这套工具生成的邮件会引用博主的具体内容：

> "看到你那款 Litter Robot 4 的测评视频，2.7M 播放，评论区 3000 条都在问好用吗——你的粉丝对这个品类是真的感兴趣..."

根据博主大小自动调整语气：
- 👤 小博主 → 轻松随性，"hey！" 开头
- 👑 大博主 → 专业简洁，直入主题

### 4️⃣ 可视化报告，告别 Excel

打开 HTML 报告，可以看到：

- 📈 关键词来源分布（哪个词找到的人最多）
- 🏆 A/B/C 分级一览
- 👤 每位达人的 bio、邮箱、代表作视频链接
- 🔍 支持筛选、搜索、排序

不用打开 Excel 也能干活 💪

## 🔄 工作流

```
产品分析 → 关键词策略 → 采集 → 评分 → 开发信
    ↓
   AI Agent 全程对话驱动
```

1. **📦 丢个亚马逊链接** — 自动提取产品信息、竞品分析、卖点提炼
2. **🔑 AI 推荐关键词组合** — 确认后开始采集
3. **🧹 自动去重 + 评分 + 分级**
4. **✉️ 一键生成个性化开发信**

## ⚡ 快速开始

```bash
git clone https://github.com/All-cookies/TikTok-KOL-Sourcing-V5.git
cd TikTok-KOL-Sourcing-V5
cp .env.example .env
# 编辑 .env，填入 TikHub API Key
npm install
```

在 Claude Code 中输入 `搜集小助手启动`，剩下的对话完成 👇
