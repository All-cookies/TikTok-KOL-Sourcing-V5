# TikTok KOL Sourcing V5

> 一款让 TikTok 红人建联更高效的工具 🎯

---

## 它能做什么

### 找达人

用关键词搜 TikTok 账号，出来的全是商家号和搬运号。

这个工具换了个思路——**先搜视频，再从视频里提取创作者** 🔍

只要丢一个亚马逊链接，AI 自动分析产品，生成四组关键词：

| 类型 | 作用 | 示例 |
|------|------|------|
| 品类词 | 找已经在做这类内容的博主 | "cat litter review" |
| 场景词 | 找目标用户常看的视频类型 | "morning routine with pets" |
| 竞品词 | 找推过竞品的博主，转化率最高 ✨ | "Litter Robot review" |
| 人群词 | 扩大范围，覆盖更多潜在人选 | "pet mom essentials" |

采集完自动去重，按播放量排序，保留最活跃的创作者。

### 筛达人

采回来的数据，系统按信号打分：

| 加分项 | 分值 | 说明 |
|--------|------|------|
| 有邮箱 | +30 | 能直接联系，不用绕路 |
| 粉丝量符合预期 | +20 | 量级匹配 |
| Bio 提到相关品类 | +15 | 内容本来就跟品牌相关 |
| 来自竞品词搜索 | +15 | 竞品已经验证过这批用户 |
| Bio 有 "collab/PR" | +10 | 明确开放合作，回复率高 |
| 活跃创作者 | +10 | 持续产出，不是弃号 |

总分从高到低分三级：
- **A级（60+）** — 直接发邮件优先联系 ✉️
- **B级（40-59）** — 先关注互动养熟
- **C级（<40）** — 备选观察

### 发邮件

邮件不是替换个名字就完事，而是**引用博主的具体视频内容**：

> "看到你那款 Litter Robot 4 的测评视频，2.7M 播放，评论区 3000 条都在问好用吗——你的粉丝对这个品类有真实需求。"

这样的邮件更有温度，不像群发模板。

还会根据博主规模自动调整语气：
- 小博主 → "hey！看到你的视频太棒了..."
- 大博主 → "Hi，Your content on X caught my attention..."

### 看结果

输出三样东西：

- **CSV** — 数据存档，方便后续管理
- **HTML 报告** — 直接用浏览器打开，可以看到所有达人的 bio、邮箱、代表作视频 🔗，支持按评级、粉丝数、播放量筛选排序
- **开发信文本** — 批量生成个性化邮件，可直接复制使用

## 快速开始

```bash
git clone https://github.com/All-cookies/TikTok-KOL-Sourcing-V5.git
cd TikTok-KOL-Sourcing-V5
cp .env.example .env
# 编辑 .env，填入 TikHub API Key
npm install
```

在 Claude Code 中输入 `搜集小助手启动`，剩下的对话完成 👇
